# Patenschaftsverwaltung

## Datenbank / Tabelle

| Spalte | Inhalt |
|--------|--------|
| Paten-ID | Fortlaufende Nummer |
| Name, Kontakt | Pate/Patin |
| Waschbär | Name, Slug |
| Stufe | Bronze/Silber/Gold |
| Startdatum | |
| Status | aktiv / gekündigt |
| Letztes Update | Datum |

## Monatliche Aufgaben

- [ ] Beiträge prüfen (Kontoauszug)
- [ ] Gold-Paten: wöchentliches Update versenden
- [ ] Silber-Paten: Fotos quartalsweise
- [ ] Neue Patenschaften: Urkunde erstellen und versenden

## Urkunde

- Vorlage mit Vereinslogo, Name Pate, Name Waschbär, Datum
- Unterschrift 1. Vorsitz
- Versand per Post oder PDF

## Kommunikation

- Bestätigung bei Aufnahme (siehe bestaetigung-vorlage.md)
- Erinnerung bei ausbleibendem Beitrag
- Kündigungsbestätigung
